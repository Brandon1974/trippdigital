# Lead Magnets Project - Comprehensive AI Prompting Guides

## Project Overview
Creating 3 professional lead magnets for Instagram Manychat automations featuring comprehensive guides on how to prompt the latest AI models (Claude, ChatGPT, Google Gemini). Each guide includes current 2026 best practices, real-world examples, and actionable templates.

**Deliverables:** 
- 3 comprehensive markdown guides (PDF-ready)
- 3 animated video versions with motion graphics
- Trippdigital.com branding on all assets

### 1. The Complete Guide to Prompting Claude (2026)
**File:** `lead-magnets/claude-prompting-guide.md`

**Content Highlights:**
- Core principle: Structure Over Cleverness
- 5 essential components (Role, Task, Context, Format, Success Criteria)
- Advanced strategies (XML tags, Sequential prompting, Context engineering)
- Real-world templates
- Best practices & common mistakes

### 2. The Complete Guide to Prompting ChatGPT (2026)
**File:** `lead-magnets/chatgpt-prompting-guide.md`

**Content Highlights:**
- Core principle: Specificity Wins Every Time
- PTCF Framework (Persona, Task, Context, Format)
- 5-sentence formula for perfect prompts
- 2026 ChatGPT superpowers (Web browsing, Code execution, DALL-E, Memory)
- Real-world prompt examples
- Template playbook (Writing, Analysis, Problem-solving)
- GPT-5.5 model features and capabilities

**Key Learning:** Vague input = Vague output. Every word spent on clarity saves 10 words of refinement.

---

### 3. The Complete Guide to Prompting Google Gemini (2026)
**File:** `lead-magnets/gemini-prompting-guide.md`

**Content Highlights:**
- Core principle: Different Models, Different Styles
- 4-part framework (Persona, Task, Context, Format)
- Model-specific prompting strategies:
  - Gemini 3.5 Flash (task-chain approach)
  - Gemini 3 (rich narrative description)
  - Veo 3.1 Lite (video/image generation)
  - Lyria 3 (music generation)
- Gemini 2026 advantages (multimodal, long context, integration)
- Real-world examples
- Pro tips for mastery

**Key Learning:** A single prompt does not work equally well across all Gemini models—each rewards different styles.
