# wiki

The structured knowledge base for Tripp Digital. Everything here has been processed from raw inputs and represents the current, accurate state of the business.

## What belongs here

- Business offers, pricing, and positioning
- Ideal customer profile (ICP) and audience research
- Brand voice, tone, and messaging guidelines
- Content strategy and platform playbooks
- Systems, tools, and automations in use
- Frequently asked questions and objections
- Partnerships and referral relationships

## Suggested files to build out

| File | Contents |
|---|---|
| `wiki/offers.md` | Products, services, pricing, bonuses |
| `wiki/audience.md` | ICP, pain points, desires, language patterns |
| `wiki/brand-voice.md` | Tone, style, phrases to use/avoid |
| `wiki/content-strategy.md` | Platforms, formats, posting cadence |
| `wiki/systems.md` | Tools, automations, workflows |
| `wiki/faq.md` | Common questions and objections with answers |
| `wiki/results.md` | Client wins, testimonials, case study notes |

## Front-matter format

Every wiki file should start with:

```
---
Source: raw/inputs/YYYY-MM-DD_filename.md
Last updated: YYYY-MM-DD
---
```

## Rules

- Wiki entries are distilled facts, not raw notes.
- When information changes, update the file and log the change in `/output/changelogs/`.
- If two wiki files contradict each other, reconcile them immediately.
