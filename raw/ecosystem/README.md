# raw/ecosystem

Stores content captured from Tripp Digital's online ecosystem: Facebook group posts, YouTube transcripts, email captures, and social content.

## What belongs here

- Facebook group posts (copied or scraped text)
- YouTube video transcripts (auto-generated or cleaned up)
- Email newsletter captures and subscriber data exports
- Comments, replies, and community discussions worth analyzing
- Competitor or industry content saved for reference

## Naming convention

```
YYYY-MM-DD_platform_description.md
```

Examples:
- `2026-06-28_facebook_ai-tools-post.md`
- `2026-07-01_youtube_transcript-lead-gen-video.md`
- `2026-07-05_email_weekly-newsletter-july-1.md`

## Rules

- Include the source URL or platform context at the top of each file.
- Do not edit the raw text — add a `---` separator and notes below if needed.
- High-value content (strong engagement, key audience language) should be flagged for wiki extraction.
